# Subindo os servicos

Executar docker compose
```console 
docker-compose up -d
```

Acessar container da aplicaçāo python
```console 
docker exec -it 5e4accc947cf bash
```

Executar aplicaçāo
```console 
python3 main.py
```

Preencher inputs
```console 
Informar host:
Informar port:
Mensagem a enviar:
```

Mensagem a enviar no formato json
```console 
{"message":"Mensagem"}
```
